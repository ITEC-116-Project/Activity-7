## Base URL
- `http://localhost:3000/api`

## Interactive Docs
- Swagger UI is available at `http://localhost:3000/docs`
    - Shows every available endpoint grouped by feature module.
    - Supports testing requests directly in the browser.
    - Reflects DTO validation rules, required fields, and sample payloads.

> Start the backend (`npm run start:dev` from the `backend` directory) and then open the Swagger link in your browser.
